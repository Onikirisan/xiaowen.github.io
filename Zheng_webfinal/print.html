Print Layout Final Project

In this project, the typefaces I used in the opening page is Ayuthaya, and I used Times New Roman in all the back pages. For the front pages of each project, I chose Noteworthy because it is cute. In addition, my project use the main tone of dark blue which can be image as the sky at night. My purpose is to show a mysterious world, so I also use a Japanese Heian period picture as the background and decorations.
